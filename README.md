# Helix

## Helix v0.5.0 — Social Core

Esta versão marca a primeira fase forte da personalidade operacional da Helix.

### Principais mudanças

- Novo núcleo de personalidade social da Helix.
- Tom mais informal, ácido e sarcástico.
- Novo `tone_router`.
- Novo `social_response_service` com reflexos sociais.
- Novo `personality_engine`.
- Melhor interpretação de comandos naturais.
- Planner de sessões de trabalho: frontend, backend, banco, estudo, pesquisa, foco e code review.
- App scanner e app registry.
- Process learning service.
- Contexto de projeto e acesso web básico.
- Correção do fluxo de confirmação de comandos.
- Proteção contra conversas casuais virarem comandos acidentais.
- Ajustes no frontend e integração com voz.

### Nota

Esta versão transforma a Helix de um assistente básico em uma presença mais operacional, contextual e personalizada dentro do projeto.
